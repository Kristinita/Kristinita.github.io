1. Описание ¶

Большой список литературы:

  1. научно-популярной,
  2. которую при желании можно бесплатно найти в Интернете,
  3. что уже использовалась или, возможно, будет при работе над Комнатами Эрика.

2. Для чего ¶

Обратите внимание на этот список, если желаете:

  1. Написать пакет для Комнат Эрика.
  2. Начать изучение какой-то темы; иметь о ней представление.
  3. Почитать захбух (научно-популярную книгу).

3. Условные обозначения ¶

Книги:

  1. Для пакетов — по которым можно написать пакет(ы) вопросов.
  2. Вспомогательные — которые, по моему мнению, полезны для развития любого, кто занимается джинджеринами. Однако они не того формата, чтобы по ним писать пакеты.
  3. NotRecommended — не рекомендуемые; причина указывается в подпапке.
  4. нвн — «нет в наличии»; у меня полной версии данной книги, но на основании информации, найденной о ней в интернете, принял решение поместить её в NotRecommended.

4. Дисклеймер ¶

Нахождение книги в списке не означает, что я её одобряю. Она может быть неформатной, не соответствовать требованиям Кристинограда (TODO: Ссылка, что это) и иметь другие недостатки.

  • Рекомендуемые книги, по которым писал пакеты вопросов, смотрите в статье «Зелёная Комната Эрика»;
  • Для информации о прочей литературе, которую хотелось бы иметь для работы над Комнатами, смотрите статью «Поиск книг для Комнат Эрика».

5. Доступ ¶

Подавляющую часть книг ниже Вы можете получить в электронном виде бесплатно, руководствуясь следующими советами.

Можно было бы сделать так, чтобы читатели могли ещё быстрее найти и скачать любую книгу из списка ниже. Однако для этого необходимо облачное хранилище объёмом не менее 100 гигабайт. На март 2020 мне известен лишь один сервис, бесплатно предоставляющий такие возможности — Degoo. Однако в Degoo нельзя расшаривать папки, производить навигацию по файлам другого пользователя. Считаю, что по состоянию на март 2020 этот сервис не упростит нахождение читателями книг.

Если у Вас имеются представления, как справиться с этой проблемой, пожалуйста напишите мне.
