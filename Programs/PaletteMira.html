1. Overview ¶

Sublime Text package, that add to command palette existing commands from third-party packages, if author(s) of them did not do this.

I include to PaletteMira commands of all known packages, that doesn't have command palette caption, not obsolete and haven't critical errors.

2. Limitations ¶

  1. You must be Sublime Text user.

3. Motivation ¶

  1. #SashaHateKeyBindings,
  2. #SashaHateContextMenu,
  3. #SashaHateNoSuricate.

4. Installation ¶

4.1. Dependencies ¶

  1. Install Suricate via Package Control.

4.2. PaletteMira ¶

Install PaletteMira, using Package Control: Add Repository with tree https://github.com/Kristinita/PaletteMira/tree/SashaDevelop.

5. Activation ¶

Ctrl+Shift+P (⌘⇧p for Mac) → Suricate: Add Profile → Select PaletteMira → Enter.

6. Usage ¶

After activation, you can run commands of packages from list. Print package name in command palette → you get result: Suricate: <Your Package>.
